# SBMF 2026 Website

Site oficial do **29th Brazilian Symposium on Formal Methods (SBMF 2026)**.

## 🚀 Como contribuir

O deploy deste site é **automático**. Qualquer alteração enviada para a branch `main` disparará o build e a atualização do site em instantes.

### Pré-requisitos
Certifique-se de ter o [Node.js](https://nodejs.org/) instalado (versão 18 ou superior).

### Comandos Rápidos
1. **Instalar dependências:**
   ```bash
   npm install

2. Teste localmente com `npm run dev`.
